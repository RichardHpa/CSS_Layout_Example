Display Property

Display Block

By Default a div is a block element. It will always start on a new line and it will stretch out to fill its parent.
Other common block elements are p, form, header, footer, section, and more.

Display:Block

Display Inline

spans are the standard inline elements. An inline element can wrap some text inside a paragraphand it won't disrupt the flow of the paragraph.
a tags are the most common inline element because you often include links within your paragraphs

Display None

Another Common element is display None. It is commonly used in javascript to be able to hide and show elements without actually having to delete or recreate them.
There is also a property called visability, the differnece between these are that display none will render the element as though it wasn't even on the page, but hidden will hide the element but it would still take some the space it would if it were fully visible.

Margin Auto

Giving a width to an element will prevent it from stretching out to the edges of its parent. Sometimes you might want this to stick int he middle of the parent instead of sticking to the edges. You could give it a set margin for each side. But if you want it to be even then give it a property of magin:0 auto. This will evenly split space from both sides to the parent.
