Margin Auto

margin: 0 auto

Giving a width to an element will prevent it from stretching out to the edges of its parent. Sometimes you might want this to stick int he middle of the parent instead of sticking to the edges. You could give it a set margin for each side. But if you want it to be even then give it a property of magin:0 auto. This will evenly split space from both sides to the parent.

Box Sizing

box-sizing: border-box;

When you add box sizing to an element, it no longer increases its size. What it does it makes the max width, including the border and padding to be the width you set. Without box sizing, the padding and borders start on the outer edge of the element, but with it they start from the inner edge and move inwards in the element.

Meida Queries

Media queries are what is required to make a website responsive. This can make it that our website looks completly different on different devices.
before you start to add media queries you need to include the following meta tag into the head of all of your html pages

<meta name="viewport" content="width=device-width, initial-scale=1">

You need to include this so your brower can figure out what is the default with of the device you are developing on.

Media Queries look like this

								
@media screen and (min-width:600px) {
	.example {
		width:100px;
		height:100px;
		background-color:red;
	}
}
@media screen and (max-width:599px) {
	.example {
		width:400px;
		height:400px;
		background-color:blue;
	}
}

What this is saying is, if the device width is bellow 599px, then have the height and width of .example is 500px and the background colour is blue.
But when the device gets to 600px or anything higher, then change the width and height to 100px and the background colour to red.
You can have as many media queries as you like and you can also combine min and max width.

								
@media screen and (min-width:600px) and (max-width:1000px){

}

This means that whatever is written in this section will only be applicable when the min width of the device is 600px, and the max width is 1000px;
